Comandos SQL Explicados

Aqui estão alguns dos comandos SQL mais comuns, com uma breve explicação de cada um.

1. CREATE DATABASE

O comando CREATE DATABASE é usado para criar um novo banco de dados. 
CREATE DATABASE nome_do_banco;

2. USE

O comando USE seleciona qual banco de dados você deseja usar em sua sessão atual. 
USE nome_do_banco;

3. CREATE TABLE

O comando CREATE TABLE é usado para criar uma nova tabela em um banco de dados. 
CREATE TABLE nome_da_tabela (
            id INT AUTO_INCREMENT,
            nome VARCHAR(100),
            idade INT,
            PRIMARY KEY(id)
        );

4. ALTER TABLE

O comando ALTER TABLE é usado para modificar a estrutura de uma tabela existente, como adicionar ou remover colunas. 
ALTER TABLE nome_da_tabela ADD coluna_exemplo VARCHAR(255);

5. DROP COLUMN

O comando DROP COLUMN é usado para remover uma coluna de uma tabela existente. 
ALTER TABLE nome_da_tabela DROP COLUMN nome_da_coluna;

6. DESCRIBE

O comando DESCRIBE fornece a estrutura de uma tabela, mostrando os nomes das colunas, tipos de dados e outras informações. 
DESCRIBE nome_da_tabela;

7. INSERT INTO

O comando INSERT INTO é usado para inserir novos dados em uma tabela. 
INSERT INTO nome_da_tabela (nome, idade) VALUES ('João', 25);

8. UPDATE

O comando UPDATE é usado para modificar dados existentes em uma tabela. 
UPDATE nome_da_tabela SET idade = 26 WHERE nome = 'João';

9. DELETE FROM

O comando DELETE FROM é usado para excluir dados de uma tabela. 
DELETE FROM nome_da_tabela WHERE nome = 'João';

10. WHERE

O comando WHERE é usado para filtrar registros com base em uma condição. 
SELECT * FROM nome_da_tabela WHERE idade > 20;

11. AND

O comando AND é usado para combinar múltiplas condições no WHERE e retornar resultados onde ambas as condições sejam verdadeiras. 
SELECT * FROM nome_da_tabela WHERE idade > 20 AND nome = 'João';

12. OR

O comando OR é usado para combinar condições e retornar resultados onde pelo menos uma das condições seja verdadeira. 
SELECT * FROM nome_da_tabela WHERE idade > 20 OR nome = 'João';

13. NOT

O comando NOT é usado para negar uma condição. 
SELECT * FROM nome_da_tabela WHERE NOT idade > 20;

14. LIKE

O comando LIKE é usado para procurar por padrões em colunas de texto. 
SELECT * FROM nome_da_tabela WHERE nome LIKE 'Jo%';

15. SELECT

O comando SELECT é usado para consultar dados de uma tabela. 
SELECT nome, idade FROM nome_da_tabela;

16. JOIN

O comando JOIN é usado para combinar dados de duas ou mais tabelas com base em uma condição comum. 
SELECT * FROM tabela1 JOIN tabela2 ON tabela1.id = tabela2.id;
